# AI Knowledge Base (Pet Project)

## Описание проекта
AI Knowledge Base - это сервис, куда пользователь загружает свои документы, а система превращает их в searchable-базу знаний. Потом пользователь задает вопрос обычным языком, а сервис отвечает на основе загруженных материалов и показывает, из каких документов взят ответ.

## Установка (Linux)
Здесь будет поэтапно описано как запустить данный сервис.

## Пример работы с данным сервисом:
1. Регистрация пользователя в системе при помощи логина и пароля. Пользователь получает свой JWT-Token, который будет использоваться для дальнейшего входа.
2. Создание коллекции или группы для документов.
3. Загрузка различных документов в формате .txt, .pdf, .md.
4. Анализ документов, разбиение на чанки.
5. Пользователь может задать вопрос AI-ассистенту, который ответит на него при помощи документов, загруженных пользователем.
6. Пользователь может уточнить из какого источника была взята информация и другие сведения доступные о документе.

## Технологии:
* Основной ЯП - Python
* REST API - FastAPI
* БД - PostgreSQL
* ORM - SQLAlchemy
* Миграция - Alembic
* Фоновые задачи - Celery
* Broker/cache - Redis
* Контейнеризация - Docker
* Reverse proxy - Nginx
* CI/CD - GitHub Actions
